# Элементы IOS

Источник: [Справочник iOS дизайна ](http://miloskiy.com/ios-design-guide-hig-na-russkom/)

|                    |                     |                                                              |                                                              |                                                              |
| ------------------ | ------------------- | ------------------------------------------------------------ | ------------------------------------------------------------ | ------------------------------------------------------------ |
| **Icons**          | Пиктограммы         | рисунок                                                      | Зачастую используется в Tab-bar                              | ![Tabbar Icons](http://miloskiy.com/wp-content/uploads/2016/01/Icons1.png) |
| **Status bar**     | Системная панель    | Содержит основную системную информацию, такую как доступность связи, текущий оператор сотовой связи, статус подключения к Wi-Fi/3g/LTE, текущее время, статус Bluetooth, будильник, заряд батареи. | Располагается в верхней части экрана                         | ![Status bar](http://miloskiy.com/wp-content/uploads/2016/01/Status1.png) |
| **Navigation bar** | Панель навигации    | Содержит в себе управляющие элементы для навигации через экраны  приложения и управления контентом текущего экрана (не всегда). <br />Элементы панели навигации: кнопка назад (левый край), заголовок текущего экрана, кнопки дополнительных действий (правый край) | Располагается в верхней части экрана под системной панелью.  | ![Navbar](http://miloskiy.com/wp-content/uploads/2016/01/NavBar2.png) |
| **Toolbar**        | Панель инструментов | Содержит набор действий для управления или взаимодействия с контентом текущего экрана. <br />Панель инструментов используется когда определенный экран требует больше чем 3х основных действий. | На iPhone располагается у нижнего края экрана. На iPad у верхнего экрана. | ![Toolbar](http://miloskiy.com/wp-content/uploads/2016/01/ToolBar3.png) |
| **Search bar**     | Панель поиска       | Вид 1: пока текст не был введен в поле поиска, там будет находится заданный по умолчанию текст (Поиск, Search).<br />Вид 2: как только пользователь начинает вводить поисковой запрос, текст Поиск» или «Search» исчезает, а кнопка очистки введенного текста появляется у правого края.<br />Вид 3: панель поиска содержит подсказку – короткое предложение которое  описывает возможности поиска. Например: «Введите название книги, автора  или часть текста для поиска». |                                                              | ![Search1](http://miloskiy.com/wp-content/uploads/2015/12/Search1.png) |
| **Tab bar**        | Панель вкладок      | Используется для быстрой навигации через разделы приложения которые  содержат в себе информацию сгруппированную по определенному признаку.<br />Панель вкладок может содержать только определенное количество вкладок (iphone - 5, ipad - 7), если число привышено то заменяется на "еще" | Располагается внизу экрана.                                  | ![Tab bar](http://miloskiy.com/wp-content/uploads/2016/01/Tab-Bar1.png) |
| **Table view**     | Таблица             | Простая таблица: может иметь заголовок(вверху таблицы) и описание (внизу таблицы). Так же может быть отображена вертикальная навигация в правой части экрана для навигации через таблицу.<br />Группированная таблица: позволяет объединитьстроки в группы.<br />Стандартная таблица: таблица содкржит изображение(может не содержать) расположенное слева и заголовок строки.<br />С подзаголовком: используется для краткого описания или объяснения.<br />С значением: позволяет отображать определенное значение которое связано с заголовком строки. |                                                              | Простая таблица:![Tables iOS](http://miloskiy.com/wp-content/uploads/2016/01/Tables1.png)<br />Группированная таблица: ![Tables iOS](http://miloskiy.com/wp-content/uploads/2016/01/Tables-3.png)<br />Стандартная таблица:![Tables](http://miloskiy.com/wp-content/uploads/2016/01/Tables-21.png)<br />Таблица с подзаголовком: ![Tables 4](http://miloskiy.com/wp-content/uploads/2016/01/Tables-4.png)<br />Таблица с значением: ![Tables 5](http://miloskiy.com/wp-content/uploads/2016/01/Tables-5.png) |
| **Activity view**  |                     | Окна такого типа используются для выполнения конкретных задач. Эти задачи могут быть системными (поделиться) или созданные разработчиком для данного приложения. |                                                              | ![image-20220213212540618](/home/as/snap/typora/49/.config/Typora/typora-user-images/image-20220213212540618.png) |
| **Actions**        | Действия            | Данный тип используется для выполнения определенного действия со списка  возможных действий или для подтверждения/отказа совершенного  пользователем действия | Отображается в виде список кнопок и должна присуствовать кнопка отмены. | ![image-20220213212825648](/home/as/snap/typora/49/.config/Typora/typora-user-images/image-20220213212825648.png) |
| **Alerts**         | Предупреждения      | Задача предупреждений заключается в уведомлении пользователя о  критической информации и, иногда, заставить (или подтолкнуть)  пользователя принять определенное решение. | Предупреждение всегда содержит заголовок и одну или две кнопки, которые предусматривают принятие решения. (например кнопки «Отправить» и «Отменить») | <img src="http://miloskiy.com/wp-content/uploads/2016/01/Alerts1.png" alt="Alerts ios" style="zoom:200%;" /> |
| **Edit menu**      | Меню редактирования | Меню редактирования позволяет пользователям выполнять функции типа «Скопировать», «Вставить», «Вырезать», когда элемент выделен. |                                                              | ![context](http://miloskiy.com/wp-content/uploads/2016/01/context.png) |
| **Popover**        | Всплывающие окна    | Их можно использовать когда определенное действие требует предварительный выбор нескольких элементов. | Данный вид всплывающего окна появляется под связанным обьектом (который  вызывает это окно) с стрелкой указывающей на этот объект. | ![Popovers](http://miloskiy.com/wp-content/uploads/2016/01/Popovers.png) |
